This is a small example of a form built with the extension Formz.

It contains some basic fields with their own validation rules; most of them are handled instantly with JavaScript, but the same rules also apply on the server-side with PHP.

It also shows the power of Formz layouts, which provide ready-to-use templates for the frameworks Twitter Bootstrap and Foundation: you can choose below which one you want to display.

You can take look at the code used by this small extension to understand how it works and how easy it is to set up a new form. You can also read Formz documentation to know everything about it (there is also a PDF version ).
